Output: Score 0-5 points + rating + detailed issues

  SectionBalanceAnalyzer → penalty (-5 to 0)

    0          → 5 pts (EXCELLENT)
    -1 to -2   → 3 pts (GOOD)

    -5         → 0 pts (POOR)
